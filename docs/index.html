🔥 What's New in v1.2

Version 1.2 introduces the main package rename to SwiftSci, Joseph form numerical stabilization for Kalman filtering, byte-level BPE tokenization, and safe DataFrame cast enhancements.

🏷️

SwiftSci Package Branding

Renamed main Swift Package to SwiftSci for clean ecosystem branding while preserving target module names (SwiftDataFrame, SwiftStats, SwiftML, SwiftForecast, etc.).

📐

Joseph Form Kalman Filter

Updated covariance matrix updates in filter() and smooth() to the Joseph form P = (I - KH) P_pred (I - KH)ᵀ + K R Kᵀ, preventing symmetry and positive-definiteness loss.

🔤

UTF-8 Byte BPE Tokenizer

Refactored BPETokenizer to split words on UTF-8 bytes mapped via GPT-2 byte-to-unicode character encoding, fixing subword tokenization for non-ASCII text.

📊

DataFrame Row Closure & Safe Cast

Added per-row closure builder addColumn(_:as:using:), fast-path type casting, and DataFrameError.partialCastFailure protection against silent data loss.

🛠 Architectural Optimizations

⚡️

OOP to Data-Oriented Design

Tree ensembles (Random Forests, GBDTs) transition from pointer-linked class instances to contiguous arrays of FlatTreeNode structures. This eliminates ARC overhead and increases CPU cache localization.

🧠

Intelligent Hardware Routing

Algorithmic branches (e.g. Decision Trees, DBSCAN) route automatically to high-IPC CPU threads, while heavy matrix mathematics route to MLX GPU-accelerated pipelines.

🚀

NaN Verification Bypass

An optional parameter checkNaN: Bool = false is provided in Stats functions to skip O(N) CPU check loops, allowing the raw hardware vector SIMD (vDSP) to run at peak capacity.

💻 Installation & Quick Start

Add SwiftAnalytics to your Package.swift dependencies:

dependencies: [
    .package(url: "https://github.com/Nodibell/SwiftAnalytics.git", from: "1.0.0")
]

Fit a GPU-accelerated Linear Regression estimator directly on a dataset:

import SwiftDataFrame
import SwiftStats
import SwiftML

// 1. Read CSV DataFrame
let df = try DataFrame.readCSV(contentsOf: csvURL)

// 2. Perform descriptive statistics (skip NaN check for speed)
let summary = try Stats.describe(df["target"].toDoubles()!, checkNaN: false)
print(summary)

// 3. Train ML model on the GPU
let regressor = LinearRegression(device: .gpu)
try await regressor.fit(features: X, targets: y)
let predictions = try await regressor.predict(features: X_test)

🚀 Core Modules Details

SwiftDataFrame

High-performance columnar data manipulation with zero-copy mappings built on Apache Arrow. Supports parallel streaming CSV parsing.

SwiftStats

Vectorized descriptive statistics, correlations, and hypothesis tests using Accelerate vDSP/LAPACK.

SwiftPreprocessing

Pipelines for feature scaling, ordinal encoding, robust normalization, and missing value indicators.

SwiftML

DOD-based Decision Trees, Random Forests, GBDTs, and SGD-based linear models on MLX.

SwiftLLM

GPU causal Transformer-decoders and Sampler interfaces powered by MLX for local AI inference. Supports zero-copy SafeTensors/GGUF parsing.

SwiftForecast

Time series forecasting (decomposition, Holt-Winters, ARIMA, SARIMA, Kalman filtering, and GARCH volatility modeling).
