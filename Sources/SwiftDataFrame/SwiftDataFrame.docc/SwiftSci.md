# ``SwiftSci``

The Scientific and Data Science Framework for Swift.

## Overview

SwiftSci is a comprehensive, production-grade ecosystem for data manipulation, machine learning, time series forecasting, statistics, NLP, and model explainability built natively in Swift.

## Topics

### Core Data & Analytics
- ``GettingStarted``
- ``MachineLearningWorkflow``
- ``TimeSeriesForecasting``

### Framework Modules
- **SwiftDataFrame**: Columnar zero-copy data frame framework.
- **SwiftML**: Supervised learning, Multi-Layer Perceptron (MLP), decision trees, and linear models.
- **SwiftCluster**: KMeans, PCA, DBSCAN, and anomaly detection.
- **SwiftForecast**: ARIMA/SARIMA, GARCH, Kalman filters, and STL decomposition.
- **SwiftStats**: Hypothesis testing, statistical distributions, and descriptive metrics.
- **SwiftPreprocessing**: Scalers, encoders, column transformers, and train/test splitting.
- **SwiftExplain**: Model explainability via KernelSHAP and feature attribution.
- **SwiftNLP**: TF-IDF, CountVectorizer, BPE tokenizer, and word embeddings.
- **SwiftOptimize**: Hyperparameter optimization and loss metrics.
- **SwiftLLM**: Transformer decoder components and prompt parsers.
